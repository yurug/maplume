Instalação no macOS

Aplicativo não assinado

O MaPlume não é assinado com um certificado Apple Developer ($99/ano). O macOS bloqueará o aplicativo por padrão.

Para executar o MaPlume após a instalação:

  1. Abra o arquivo .dmg baixado
  2. Arraste o MaPlume para a pasta Aplicativos
  3. Abra o Terminal (Aplicativos → Utilitários → Terminal)
  4. Execute este comando:
xattr -cr /Applications/MaPlume.app

Agora você pode executar o MaPlume normalmente.

Quer ajudar? Se usuários suficientes apoiarem o projeto, poderemos obter um certificado Apple e eliminar esta etapa.

Instalação no Linux

O MaPlume é distribuído como AppImage, um formato portátil que funciona na maioria das distribuições Linux.

Para instalar:

  1. Baixe o arquivo .AppImage
  2. Torne-o executável:
chmod +x MaPlume-1.0.2.AppImage
  1. Dê duplo clique no arquivo ou execute pelo terminal:
./MaPlume-1.0.2.AppImage

Dica: Você pode mover o AppImage para ~/.local/bin/ para acessá-lo de qualquer lugar.
