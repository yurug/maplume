macOSへのインストール

未署名アプリケーション

MaPlumeはApple開発者証明書（年間$99）で署名されていません。macOSはデフォルトでアプリをブロックします。

インストール後にMaPlumeを起動するには：

  1. ダウンロードした.dmgファイルを開く
  2. MaPlumeをアプリケーションフォルダにドラッグ
  3. ターミナルを開く（アプリケーション → ユーティリティ → ターミナル）
  4. このコマンドを実行：
xattr -cr /Applications/MaPlume.app

これでMaPlumeを通常どおり起動できます。

ご協力いただけますか？十分なユーザーがプロジェクトをサポートしてくださると、Apple証明書を取得してこのステップを省略できます。

Linuxへのインストール

MaPlumeはAppImage形式で配布されています。ほとんどのLinuxディストリビューションで動作するポータブル形式です。

インストール手順：

  1. .AppImageファイルをダウンロード
  2. 実行可能にする：
chmod +x MaPlume-1.0.2.AppImage
  1. ファイルをダブルクリックするか、ターミナルから実行：
./MaPlume-1.0.2.AppImage

ヒント：AppImageを~/.local/bin/に移動すると、どこからでもアクセスできます。
