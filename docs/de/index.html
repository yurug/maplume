Installation auf macOS

Nicht signierte Anwendung

MaPlume ist nicht mit einem Apple-Entwicklerzertifikat signiert ($99/Jahr). macOS blockiert die App standardmäßig.

Um MaPlume nach der Installation zu starten:

  1. Öffnen Sie die heruntergeladene .dmg-Datei
  2. Ziehen Sie MaPlume in den Programme-Ordner
  3. Öffnen Sie das Terminal (Programme → Dienstprogramme → Terminal)
  4. Führen Sie diesen Befehl aus:
xattr -cr /Applications/MaPlume.app

Jetzt können Sie MaPlume normal starten.

Möchten Sie helfen? Wenn genügend Nutzer das Projekt unterstützen, können wir ein Apple-Zertifikat erwerben und diesen Schritt entfernen.

Installation auf Linux

MaPlume wird als AppImage verteilt, ein portables Format, das auf den meisten Linux-Distributionen funktioniert.

Zur Installation:

  1. Laden Sie die .AppImage-Datei herunter
  2. Machen Sie sie ausführbar:
chmod +x MaPlume-1.0.2.AppImage
  1. Doppelklicken Sie auf die Datei oder führen Sie sie im Terminal aus:
./MaPlume-1.0.2.AppImage

Tipp: Sie können das AppImage nach ~/.local/bin/ verschieben, um von überall darauf zuzugreifen.
